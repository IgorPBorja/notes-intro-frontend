Listas

Uma lista não-ordenada ( unordered list)usa a tag ul, e cada elemento da lista deve ser inserido em uma tag li.

  • A
  • B
  • C

Por outro lado, em uma lista ordenada (ordered list) apresenta seus elementos numerados, e utiliza a tag ol. Cada elemento da lista, novamente, deve ser inserido em uma tag li.

  1. A
  2. B
  3. C

Listas aninhadas (nested lists)

  • Lista 1
    • A
    • B
    • C
  • Lista 2
    1. X
    2. Y
    3. Z
    4. W

Tabelas

As tabelas são criadas através da tag table. Cada linha (row) é uma tag tr. Podemos também ter um header (primeira linha), com a tag th. Dentro de cada row, registramos uma entrada como um elemento da tag td, table data. Por exemplo:

Coluna 1 Coluna 2 Coluna 3 Coluna 4
1 2 3 4
5 6 7 8

Formulários

Um formulário é criado com a tag form. Cada entrada de dados é um objeto da tag input, com os seguintes atributos:

  • type: tipo de dado (default = text)
  • name: texto a aparecer para identificar aquela entrada
  • id: identificador

Podemos colocar o botão para submissão de formulário como um objeto da tag button, de tipo submit.

Para mostrar texto associado a determinada entrada no formulário, precisamos de um label, cujo atributo for seja o id daquele input. Ademais, podemos, utilizando o atributo placeholder, exibir ao usuário um texto prévio, que aparece antes de ele clicar naquele elemento de input.

Por fim, com o atributo value, podemos atribuir um valor fixo (hardcoded value), para, por exemplo, propósitos de teste. Esse valor aparece já preenchido quando o formulário aparece pela primeira vez na tela, e precisa ser apagado para que outro dado seja submetido no lugar.






Podemos também ter áreas onde o usário pode inserir textos maiores, com a tag textarea

Além disso, podemos ter botões para escolher determinado valor com o tipo radio da tag input. Inputs do tipo "radio" com o mesmo nome (mesmo valor do atributo name) estão interligados: somente um pode ser escolhido. O valor a ser transmitido, se um determinado radiobutton for escolhido, é o valor do atributo value.

Área de texto:


Escolha uma das opções a seguir:

A B C

Outras opções, independentes dos botões acima:

D E

Por outro lado, com checkboxes, (tags input com type="checkbox"), podemos múltiplos valores simultaneamente. Podemos também atribuir um valor padrão: se adicionarmos o atributo checked, a caixa aparece selecionada por pardrão.

A B C

Selecionando entre múltiplas opções

Dentro de formulários, usando a tag select, podemos implementar dropdown menus, isto é, menus que descem, revelando opções de escolha.



Cada valor de cada radiobutton escolhido pelo usuário é salvo na URL, da forma "&name_of_radiobutton=chosen_value".
